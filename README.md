# DOit - Track your progress
> A To-Do application written in kotlin using Android Architectural components with MVVM pattern 

 - **Kotlin** - as a programming language.
 - **Koin** - Dependency injection
 - **Coroutines** - To query db in worker thread.
 - **ViewModel** - Used to manage UI related data, Survives config changes.
 - **LiveData** - Lifecycle aware Observable data holder class.
 - **Room** - To persist user todo records
 - **AndroidX** - Project uses androidx packages.

#### Functionalities included:
- Create Todo
- Delete existing Todo
- View/Update Todo
- Search Todo list

#### Concepts included:
- Material Design Guidelines
- MVVM Architecture
- Repository Pattern
- Single Activity Concept
- Loading Animations
- Diff Utils
- Swipe to delete
- Pull to refresh
